# Poisson Bench
Just for fun :)

A single-file, in-browser football betting model (**Double Poisson + Dixon-Coles**). Fit team strengths from past results, see the full scoreline distribution, and compare model probabilities against bookmaker odds to find expected value (EV).

> ⚠️  Output is only as good as your λ, and a hand-tuned model won't beat a bookmaker. 

**Use it:** (1) Paste recent results and click **Fit**, pick home/away, **Apply to model** — or set λ by hand. (2) Type the bookmaker's **American odds** per market. (3) Green **+EV** = positive expected value; red **SKIP** = negative.

Bilingual (EN / 中文, one-click toggle). No backend, no build — just open `index.html`. MIT licensed.

# 摸鱼实验室🖐️🐟️
摸鱼作品，做着玩的。

单文件、纯浏览器运行的足球投注模型(**双泊松 + Dixon-Coles**)。从历史结果拟合球队强度，查看完整比分分布，把模型概率与庄家赔率对比，得出期望值(EV)。

> ⚠️ 输出可靠性取决于你的 λ，手调模型跑不赢庄家。

**用法：**(1)粘贴近期结果，点「拟合」，选主客队，「代入模型」。或手动设 λ。(2)各盘口填庄家**美式赔率**。(3)绿色 **+EV** = 正期望；红色 **SKIP** = 负期望。

中英双语一键切换。无后端、无构建，打开 `index.html` 即用。MIT 许可。
